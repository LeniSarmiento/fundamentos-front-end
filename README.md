# Fundamentos Front-End - UCAL

Repositorio del curso de Fundamentos de Front-End.

## Como contribuir

1. Haz un **Fork** de este repositorio (boton "Fork" arriba a la derecha)
2. Clona tu fork en tu computadora
3. Crea tu archivo HTML con tu nombre (ejemplo: `juan-perez.html`)
4. Sube tus cambios a tu fork
5. Crea un **Pull Request** hacia este repositorio

## Reglas

- Solo archivos HTML
- Nombra tu archivo con tu nombre (ejemplo: `maria-garcia.html`)
- No modifiques los archivos de otros companeros
